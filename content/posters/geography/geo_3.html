ARDUINO ДЕГЕНІМІЗ НЕ?WHAT IS ARDUINO?ЧТО ТАКОЕ ARDUINO?

Arduino – бұл электронды жобаларды жасауға арналған ашық бастапқы кодты электроника платформасы. Ол аппараттық (басқарма платасы) және бағдарламалық (IDE) бөліктерден тұрады. Arduino is an open-source electronics platform for building electronic projects. It consists of a physical programmable circuit board (microcontroller) and a piece of software (IDE). Arduino — это открытая платформа для электроники, используемая для создания электронных проектов. Она состоит из физической программируемой платы (микроконтроллера) и программного обеспечения (IDE).

ҚАЛАЙ ЖҰМЫС ІСТЕЙДІ?HOW DOES IT WORK?КАК ЭТО РАБОТАЕТ?

Сіз Arduino IDE бағдарламасында код жазасыз (скетч деп аталады). Бұл код Arduino платасына жүктеледі. Плата осы нұсқауларды орындап, датчиктерден сигналдарды қабылдап, моторларды немесе жарықдиодтарды басқарады. You write code (called a "sketch") in the Arduino IDE. This code is uploaded to the Arduino board. The board then executes these instructions, taking input from sensors and controlling outputs like motors or LEDs. Вы пишете код (называемый "скетчем") в Arduino IDE. Этот код загружается на плату Arduino. Затем плата выполняет эти инструкции, принимая ввод от датчиков и управляя такими выходами, как двигатели или светодиоды.

Бұл қалай жұмыс істейді?How it works:Как это работает:

Arduino-ның 13-ші пиніне жарықдиодты (резистор арқылы) қосыңыз. Код жарықдиодты бір секундқа қосып, бір секундқа өшіріп, циклде қайталайды. Connect an LED (with a resistor) to pin 13 of your Arduino. The code tells the LED to turn on for one second, then off for one second, repeating in a loop. Подключите светодиод (через резистор) к 13-му пину Arduino. Код заставит светодиод включаться на одну секунду, затем выключаться на одну секунду, повторяя это в цикле. 


void setup() {
  pinMode(13, OUTPUT); // 13-ші пинді шығыс ретінде орнатуSet pin 13 as an outputНастроить пин 13 как выход
}

void loop() {
  digitalWrite(13, HIGH); // Жарықдиодты қосу (HIGH - 5V)Turn the LED on (HIGH is 5V)Включить светодиод (HIGH - 5В)
  delay(1000);            // Бір секунд күтуWait for a secondЖдать одну секунду
  digitalWrite(13, LOW);  // Жарықдиодты өшіру (LOW - 0V)Turn the LED off (LOW is 0V)Выключить светодиод (LOW - 0В)
  delay(1000);            // Бір секунд күтуWait for a secondЖдать одну секунду
}
