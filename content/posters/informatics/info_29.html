1. Алгоритмнің қасиеттері

1. Properties of Algorithms

1. Свойства алгоритма

1. Дискреттілік (Discreteness)

Алгоритм жеке-жеке қадамдарға бөлінуі керек.

2. Анықтылық (Definiteness)

Әрбір команда орындаушыға түсінікті болуы тиіс.

3. Нәтижелілік (Effectiveness)

Алгоритм белгілі бір қадамнан кейін нәтижеге жетуі керек.

4. Жалпылық (Mass Character)

Алгоритм бір типтегі есептердің барлығына жарауы керек.

3. Алгоритм түрлері

3. Types of Algorithms

3. Типы алгоритмов

Сызықтық / Linear

Әрекеттер тізбектей орындалады.
Actions perform sequentially.

Тармақталған / Branching

Шартқа байланысты таңдау.

Циклдік / Cyclic

Әрекеттер қайталанады.
Actions repeat.

5. Программалау ортасы (IDE)

5. Programming Environment (IDE)

5. Среда программирования (IDE)

IDE (Integrated Development Environment) – программаларды құруға арналған кешенді құрал. Ол редактор, компилятор және жөндеушіні біріктіреді.

IDE is a comprehensive tool for building software. It combines an editor, compiler, and debugger.

IDE (Интегрированная среда разработки) – это комплексный инструмент для создания программ. Она объединяет редактор, компилятор и отладчик.

ЗАПОМНИТЕ

Алгоритм – бұл жоспар. Блок-схема – жоспардың суреті. Программа – компьютерге арналған жоспар.

An algorithm is a plan. A flowchart is a picture of the plan. A program is a plan for a computer.

Алгоритм – это план. Блок-схема – это рисунок плана. Программа – это план для компьютера.
