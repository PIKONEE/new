1. Басқару жүйесі

1. Control System

1. Система управления

Басқару – бұл белгілі бір мақсатқа жету үшін объектіге әсер ету. Кез келген басқару жүйесі екі бөліктен тұрады: Басқарушы объект және Басқарылушы объект.

Control is influencing an object to achieve a goal. Any control system consists of two parts: the Controlling Object and the Controlled Object.

Управление – это воздействие на объект для достижения цели. Любая система управления состоит из двух частей: Управляющий объект и Управляемый объект.

2. Кері байланыс (Тұйық жүйе)

2. Feedback (Closed Loop)

2. Обратная связь (Замкнутая система)

Басқару тиімді болуы үшін, басқарушы объект нәтижені біліп отыруы керек. Бұл ақпарат ағыны кері байланыс деп аталады.

For control to be effective, the controller needs to know the result. This flow of information is called feedback.

Чтобы управление было эффективным, управляющий должен знать результат. Этот поток информации называется обратной связью.

3. Бағдарламалау – басқару құралы

3. Programming as a Tool for Control

3. Программирование – инструмент управления

Компьютерді немесе роботты басқару үшін біз оларға түсінікті тілде нұсқаулар (алгоритмдер) жазамыз. Бұл – бағдарламалау.

To control a computer or robot, we write instructions (algorithms) in a language they understand. This is programming.

Чтобы управлять компьютером или роботом, мы пишем инструкции (алгоритмы) на понятном им языке. Это – программирование.

ЗАПОМНИТЕ

Басқару – бұл ақпарат алмасу. Кері байланыс басқарудың дұрыстығын тексереді. Бағдарламалау – компьютерді басқару тілі.

Control is information exchange. Feedback checks if control is correct. Programming is the language of controlling computers.

Управление – это обмен информацией. Обратная связь проверяет правильность управления. Программирование – это язык управления компьютером.
