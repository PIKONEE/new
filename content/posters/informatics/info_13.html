1. Орындаушы дегеніміз кім?

1. Who is an Executor?

1. Кто такой Исполнитель?

Орындаушы – белгілі бір командаларды түсінетін және орындай алатын объект (адам, робот, компьютер, жануар).

An Executor is an object (human, robot, computer, animal) that can understand and execute specific commands.

Исполнитель – это объект (человек, робот, компьютер, животное), который понимает и может выполнять определенные команды.

2. Орындаушының командалар жүйесі (ОКЖ)

2. Executor's Command System (ECS)

2. Система команд исполнителя (СКИ)

Орындаушы тек өзі білетін командаларды ғана орындай алады. Бұл командалар жиынтығы – СКИ деп аталады.

An executor can only perform commands it "knows". This set of commands is called the Command System.

Исполнитель может выполнять только те команды, которые он "знает". Этот набор команд называется СКИ.

Робот "Пицца пісір" командасын түсінбейді, себебі ол оның СКИ-інде жоқ.

The robot doesn't understand "Cook Pizza" because it's not in its Command System.

Робот не понимает команду "Испеки пиццу", так как её нет в его СКИ.

REMEMBER

ЗАПОМНИТЕ

Орындаушы командаларды ғана орындайды. Егер команда СКИ-де жоқ болса, орындаушы оны түсінбейді.

An executor only performs commands. If a command is not in the ECS (Command System), the executor won't understand it.

Исполнитель выполняет только команды. Если команды нет в СКИ, исполнитель её не поймет (возникнет ошибка).
