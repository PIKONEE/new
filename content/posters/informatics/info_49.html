1. What is Vector Graphics?

1. Что такое векторная графика?

Векторлық графика – кескіндерді пиксельдер емес, геометриялық примитивтер (нүкте, сызық, шеңбер) арқылы сипаттайтын графика түрі.

Vector graphics describes images using geometric primitives (points, lines, circles) rather than pixels.

Векторная графика – это вид графики, где изображение описывается не пикселями, а геометрическими примитивами (точками, линиями, кругами).

Бұл суреттер компьютер жадында формула ретінде сақталады.

These images are stored in computer memory as formulas.

Эти изображения хранятся в памяти компьютера в виде формул.

2. Сапа және Масштабтау

2. Quality and Scaling

2. Качество и Масштабирование

Векторлық графиканың басты артықшылығы – суретті қанша үлкейтсең де, сапасы бұзылмайды.

The main advantage of vector graphics is that you can scale the image indefinitely without losing quality.

Главное преимущество векторной графики – изображение можно увеличивать сколько угодно, и качество не испортится.

ЗАПОМНИТЕ

Векторлық графика = Формулалар. Ол сапасын жоғалтпайды, файл өлшемі кіші, бірақ фотосуреттер үшін жарамайды.

Vector graphics = Formulas. It doesn't lose quality, has small file size, but is not suitable for photos.

Векторная графика = Формулы. Она не теряет качества, имеет малый размер файла, но не подходит для фотографий.
