Oxides - Compounds of Oxygen!

Оксиды - соединения кислорода!

XₓOᵧ - екі элементтен, оттегі болатын құрамдас. Негіздік (CaO), қышқылдық (CO₂), өндіктік (Al₂O₃). Суға реакциясы: базис немесе қышқыл түзеді! ОКСИДТЕР БАРЛЫҚ ЖЕРДЕ!

XₓOᵧ - two elements, one is oxygen. Basic (CaO), acidic (CO₂), amphoteric (Al₂O₃). Reactions with water form bases or acids! OXIDES EVERYWHERE!

XₓOᵧ - два элемента, один кислород. Основные (CaO), кислотные (CO₂), амфотерные (Al₂O₃). Реакции с водой образуют основания или кислоты! ОКСИДЫ ВЕЗДЕ!

2. ОКСИДТЕРДІҢ РЕАКЦИЯЛАРЫ

2. REACTIONS OF OXIDES

2. Реакции оксидов

✅ НЕГІЗДІК ОКСИДТЕР СУҒА:

CaO + H₂O → Ca(OH)₂ (құм буын)

Na₂O + H₂O → 2NaOH (натрий гидроксид)

✅ BASIC OXIDES + WATER:

CaO + H₂O → Ca(OH)₂ (slaked lime)

Na₂O + H₂O → 2NaOH (sodium hydroxide)

✅ ОСНОВНЫЕ ОКСИДЫ + ВОДА:

CaO + H₂O → Ca(OH)₂ (гашеная известь)

Na₂O + H₂O → 2NaOH (гидроксид натрия)

✅ ҚЫШҚЫЛДЫҚ ОКСИДТЕР СУҒА:

CO₂ + H₂O → H₂CO₃ (углеводородты қышқыл)

SO₃ + H₂O → H₂SO₄ (күріш қышқылы)

✅ ACIDIC OXIDES + WATER:

CO₂ + H₂O → H₂CO₃ (carbonic acid)

SO₃ + H₂O → H₂SO₄ (sulfuric acid)

✅ КИСЛОТНЫЕ ОКСИДЫ + ВОДА:

CO₂ + H₂O → H₂CO₃ (угольная кислота)

SO₃ + H₂O → H₂SO₄ (серная кислота)

✅ ОКСИДТЕРДІҢ НЕГІЗДІК + ҚЫШҚЫЛДЫҚ РЕАКЦИЯСЫ:

CaO + CO₂ → CaCO₃ (кальцийге карбонат)

MgO + CO₂ → MgCO₃ (магниумге карбонат)

✅ BASIC + ACIDIC OXIDES:

CaO + CO₂ → CaCO₃ (calcium carbonate)

MgO + CO₂ → MgCO₃ (magnesium carbonate)

✅ ОСНОВНЫЕ + КИСЛОТНЫЕ ОКСИДЫ:

CaO + CO₂ → CaCO₃ (карбонат кальция)

MgO + CO₂ → MgCO₃ (карбонат магния)

ЗАКЛЮЧЕНИЕ

Оксидтер (XₓOᵧ) - екі элементтен құрылған, оттегі болатын заттар. Үш түрі: негіздік (CaO, MgO), қышқылдық (CO₂, SO₃), өндіктік (Al₂O₃). Суға реакциясы: негіздік өндіктігі негіз түзеді, қышқылдық оттегінің қышқыл түзеді. Оксидтер барлық технологияның негіздері!

Oxides (XₓOᵧ) - compounds of two elements, one oxygen. Three types: basic (CaO, MgO), acidic (CO₂, SO₃), amphoteric (Al₂O₃). Water reactions: basic form bases, acidic form acids. Oxides are foundation of all technology!

Оксиды (XₓOᵧ) - соединения двух элементов, один кислород. Три типа: основные (CaO, MgO), кислотные (CO₂, SO₃), амфотерные (Al₂O₃). Реакции с водой: основные образуют основания, кислотные - кислоты. Оксиды - основа всей технологии!
