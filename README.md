# Проект "Интерактивные Плакаты"

Это десктоп-приложение для школ, предоставляющее доступ к интерактивным учебным материалам.

## Как запустить и собрать проект

### Шаг 1: Установка зависимостей

Перед первым запуском или сборкой необходимо установить все требуемые библиотеки.
Откройте терминал (командную строку) в папке проекта и выполните команду:

```bash
pip install -r requirements.txt
